Nintendo 3DS の Web ブラウザで遊べるリバーシです。  
CPUとの対戦、１台で２人対戦が可能です。  
JavaScript + DOM で動いています。  
MIT ライセンスです。  
  
アプリURL: [http://okanoworld.appspot.com/reversi3ds](http://okanoworld.appspot.com/reversi3ds)
